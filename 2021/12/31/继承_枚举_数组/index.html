js中的继承 枚举和数组介绍

继承 枚举 数组

圣杯模式：解决互相影响的问题

命名空间：用对象解决变量重名，模块化开发。

属性的表示方法：

obj.属性名
obj.[“属性名”]

遍历枚举

遍历对象的属性：

在枚举中调用对象属性不能用 对象名.属性名 ,可以用对象名[属性名]

A instanceof B 看A对象的原型链上有没有 B的原型

typeof可能返回的类型：

  • string
  • object
  • undefined
  • function
  • number
  • boolean

this

  1. 函数预编译过程 this - > window
  2. 全局作用域里 this -> window
  3. call / apply 可以改变函数运行时的this指向
  4. obj.func(); func() 里面的this指向obj

浅层克隆（引用值不独立，原始值独立）

深层克隆（递归）

定义方式：

数组字面量：var 数组名 = [ ];

构造方法： var 数组名 = new Array(参数);

参数个数为1时代表数组的长度（只能为整数），个数大于1时代表数组的具体内容。

数组的所有方法全来自 Array.prototype

数组可以越界读取数据（undefined）

数组的方法：

1. 改变原数组：

  • push : 在数组最后面添加内容（可一次添加多个）
  • pop:剪切出数组最后一位
  • unshift:在数组最前面加内容(可一次添加多个)
  • shift:剪切数组最前面的内容
  • reverse:数组逆序
  • sort：将数组内容排序(按ASCII码排序)
正常使用：

规则：

  1. 必须写俩形参
  2. 看返回值：
    1. 返回值为负数：前一个数放在前面
    2. 返回值为正数：后一个数放在前面
    3. 返回值为0时：顺序不动
  • splice:数组切片(可在任意位置添加内容)
    数组名.splice(从第几位开始，截取的长度，在切口处添加新的数据)

    2.不改变原数组：

  • concat() :连接两个数组
  • slice : 截取
    数组.slice(截取开始位置，截取结束位置)
  • join：将数组各位连接起来
    数组名.join(字符串)；
    无参数默认为用 , 连接
  • split:将字符串按照某一规则拆分成数组,与join互逆 类数组：像数组的对象
    属性要为索引（数字）属性，必须有length属性，最好加上push
    arguments是个类数组

在try里面的发生错误，不会执行错误后的try里的代码

e.name:错误名称
e.message:错误信息

Error.name的六种值对应的信息：

  1. EvalError : eval()的使用与定义不一致
  2. RangeError：数值越界
  3. ReferenceError：非法或不能识别的引用数值
  4. SyntaxError：发生语法解析错误
  5. TypeError：操作数类型错误
  6. URIError：URI处理函数使用不当

es5.0严格模式：

不再兼容es3的一些不规则语法，使用全新的es5规范（不会对不兼容严格模式的浏览器产生影响）

“use strict”;来启动es5.0严格模式（写在逻辑最顶端）
可以使用全局严格模式
也可以使用局部函数内严格模式

可以将with里面的代码的作用域链最顶端改为新给出的对象（降低效率）
